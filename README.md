# udp-opudp

**OPUDP** — ZIVPN UDP Ultimate Management Panel  
Author: [OfficialOnePesewa](https://github.com/OfficialOnePesewa)

## Install

\`\`\`bash
wget -O i.sh https://raw.githubusercontent.com/OfficialOnePesewa/udp-opudp/main/install.sh && sudo chmod +x i.sh && sudo ./i.sh
\`\`\`

Then run the panel:

\`\`\`bash
opudp
\`\`\`

## ZIVPN App Config

After adding a user via `opudp → option 6`:

| Field | Value |
|---|---|
| UDP Server | Your VPS IP |
| UDP Password | Password you set in the panel |

> Username is for the panel only — only the **password** connects the app.
